1、采用`kebab-case`命名的：

- 文件夹
- 单文件组件
- 组件在html模板中使用（`<my-component></my-component>`）
- 在模板中prop传入属性到子组件（`<my-componnet set-text="hello"/>`）
- 所有事件名（`this.$emit('api-reload')`）

2、采用`PascalCase`命名：

- 公共基础组件（`MfcSelect`）
- js中components注册组件时（`import MyComponent from './my-component.vue'`）
- 组件的name属性（`name: 'MyComponent'`）

3、采用`camelCase` 命名：

- 子组件接收prop属性

```
props: {
  setText: String
}

```
